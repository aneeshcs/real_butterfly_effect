The Real Butterfly Effect

Interactive research notebooks for studying predictability in geophysical turbulence, using pseudospectral quasi-geostrophic models. Each notebook runs entirely in your browser via WebAssembly — no installation required.

2D QG

2D Quasi-Geostrophic Turbulence

Pseudospectral solver for the 2D QG equations on a doubly-periodic domain:

∂q/∂t + J(ψ,q) = ν(−1)^(p+1) ∇^(2p) q

where q = ∇²ψ + βy is the potential vorticity.

  • RK4 time stepping, 2/3-rule dealiasing
  • Energy & enstrophy evolution
  • KE spectra with k⁻³ / k⁻⁵/³ reference slopes
  • Twin-run predictability experiment
  • Finite-time Lyapunov exponent (FTLE)

3D QG · 2-Layer

2-Layer Quasi-Geostrophic Turbulence

Pseudospectral solver for the 2-layer QG model representing stratified 3D QG flow:

q_n = ∇²ψ_n + F_n(ψ_{3-n} − ψ_n) + βy

with barotropic/baroclinic decomposition.

  • Adjustable stratification F₁, F₂ and bottom drag κ
  • Barotropic & baroclinic KE spectra
  • Layer-wise energy & enstrophy
  • Twin-run predictability experiment
  • Layer-wise FTLE

SQG

Surface Quasi-Geostrophic Turbulence

Pseudospectral solver for the SQG equations, where surface buoyancy drives the flow via a shallower Green's function:

∂b/∂t + J(ψ,b) = ν(−1)^(p+1) ∇^(2p) b

with ψ̂ = b̂ / |k| (vs. b̂ / k² in 2D QG).

  • Filamentary surface buoyancy structures
  • KE spectrum E(k) ~ k⁻⁵/³ (Lorenz 1969 regime)
  • Surface energy = buoyancy variance (SQG identity)
  • Twin-run predictability experiment
  • Finite-time Lyapunov exponent (FTLE)
